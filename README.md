# 🛒 Northwind Database – SQL Analysis

## 📌 Project Overview
This project analyzes the Northwind database using SQL to answer important business questions related to sales, customers, and products.

## 🎯 Objectives
- Analyze product sales.
- Find top customers by revenue.
- Identify monthly sales trends.
- Analyze category-wise sales.
- Count customer orders.

## 🛠 Tools Used
- SQLite
- DB Browser for SQLite
- Visual Studio Code
- Git & GitHub

## 📂 Files Included
- `queries.sql` – SQL queries used for analysis.
- `analysis.ipynb` – Python notebook for data analysis.
- `README.md` – Project documentation.
- `northwind.db` – Northwind SQLite database.

## 📊 SQL Queries Performed
1. Top 10 best-selling products.
2. Top 10 customers by total revenue.
3. Monthly sales analysis.
4. Category-wise sales performance.
5. Customer order count.

## ✅ Conclusion
This project demonstrates how SQL can be used to extract meaningful business insights from the Northwind database by analyzing sales, customers, and product performance.
